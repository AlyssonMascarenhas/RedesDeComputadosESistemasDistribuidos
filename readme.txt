Para executar, abra 3 terminais:

Terminal 01 : vá na pasta onde estão os arquivos execute "javac *.java", apos isso, execute ainda no memso terminal "rmiregistry".

Terminal 02 : vá na pasta onde estão os arquivos e execute o Servidor. "java Servidor".

Terminal 03 : vá na pasta onde estão os arquivos e execute o cliente. "java Main".
